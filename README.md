# Introduction
This game is a match-3 type game that I got inspired to recreate when a friend of mine showed me a [similar game](https://www.youtube.com/watch?v=Q01iYD3K5Xo) on WeChat. The rules of the game are simple, match tiles in groups of 3 until the level is cleared to win. If the tile deck is filled with 7 tiles without a group of 3 matches, the game ends.

# How to Run
### Windows
Run `main.exe`

### Linux
Install `wine` using your distro's package manager. For example, for Ubuntu:
```
sudo apt install wine
```
To run the game:
```
wine main.exe
```
